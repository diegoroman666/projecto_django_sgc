Acerca del Sistema

El Sistema de Gestión de Correspondencias está diseñado para agilizar y organizar el flujo de documentos dentro de la organización. Permite registrar, listar, buscar y analizar las correspondencias recibidas y enviadas. Además, proporciona estadísticas e historial de operaciones realizadas, facilitando una gestión eficiente y ordenada.

Entre las funcionalidades principales se encuentran:

  • Registro de correspondencias
  • Listado y búsqueda avanzada
  • Importación de archivos CSV
  • Visualización de estadísticas
  • Historial de operaciones
